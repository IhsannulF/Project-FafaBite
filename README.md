# 🍔 FafaBite - Food Rescue Application

FafaBite adalah aplikasi Android inovatif yang mengusung gerakan #AntiFoodWaste. Kami menghubungkan Mitra Restoran yang memiliki makanan berlebih (surplus) di penghujung hari dengan Konsumen yang ingin menikmati makanan lezat dengan harga yang jauh lebih hemat.

## ✨ Fitur Utama

**👤 Untuk Konsumen (Pembeli):**
* **Eksplorasi Makanan:** Cari makanan surplus dari restoran terdekat.
* **Flash Sale & Diskon:** Dapatkan potongan harga mulai dari 50% untuk makanan berkualitas.
* **Kategori Cerdas:** Filter pencarian berdasarkan jenis makanan (Nasi, Roti & Kue, Dessert, dll).
* **Sistem Pick-up:** Pesan di aplikasi, dan ambil langsung ke toko dengan menunjukkan bukti pesanan/QR.

**🏪 Untuk Mitra (Restoran/Pedagang):**
* **Dashboard Penjualan:** Pantau performa stok surplus dan pendapatan hari ini secara real-time.
* **Manajemen Menu:** Tambahkan stok makanan sisa dengan cepat dan atur harga diskonnya.
* **Manajemen Pesanan:** Terima pesanan masuk dan verifikasi pengambilan dengan fitur Scan QR.
* **Kendali Toko:** Atur status buka/tutup toko hanya dengan satu tombol *switch*.

## 🛠️ Teknologi yang Digunakan
* **Frontend:** Android Native (Kotlin), XML Layouts, Material Design
* **Arsitektur:** RESTful API Integration menggunakan Retrofit
* **Desain UI/UX:** Custom Dialogs, Bottom Navigation, Interactive Cards
* **Lainnya:** GSON Converter
